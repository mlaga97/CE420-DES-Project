# CE420-DES-Project
Implementation of DES in C with C++ test application.

# Instructions for use (Linux)
```
# Clone
git clone https://github.com/mlaga97/CE420-DES-Project
cd CE420-DES-Project

# Build
g++ main.cpp des/*.cpp -o des.exe

# Run
./des.exe
```
